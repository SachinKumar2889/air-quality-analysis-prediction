# Air Quality Analysis and Prediction using Machine Learning

## Overview
This project analyzes air quality data from major Indian cities and predicts PM2.5 concentration using Linear Regression.

## Objectives
- Analyze air pollution trends
- Classify AQI categories
- Identify pollution hotspots
- Perform season-wise analysis
- Predict PM2.5 concentration

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Model Performance
- R² Score: 0.755
- MAE: 14.01
- MSE: 515.29

## Features
- AQI Classification
- Pollution Hotspot Ranking
- Health Risk Assessment
- Season-wise Pollution Analysis
- PM2.5 Prediction
- Air Quality Alert System

## Conclusion
Delhi recorded the highest pollution levels, while winter showed the highest average PM2.5 concentration. The model successfully predicts PM2.5 and provides health recommendations based on predicted pollution levels.
